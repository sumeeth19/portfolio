1. Problem Statement

Traditional cricket statistics such as total runs and batting average do not fully capture a player's consistency, impact, and performance under different match conditions. The objective was to transform raw innings-level data into deeper analytical insights that explain how and when performance changes.

2. Project Objective

  • Analyze consistency and impact across innings
  • Evaluate performance under different conditions
  • Identify trends, volatility, and future patterns
  • Present insights using business-style dashboards

3. Dataset Overview

  • Innings-level dataset across formats (ODI, T20, IPL)
  • Core fields: Runs, Balls, Strike Rate, Dismissal, Venue
  • Derived attributes: Home/Away/Neutral, Year, Match Context
  • Data size: ~270+ innings

4. Data Preparation & Feature Engineering

  • Standardized match dates and innings order
  • Created milestone flags (30+, 50+, 100+)
  • Derived contextual tags (Chase vs Set target)
  • Calculated rolling averages and cumulative metrics

6. Key Metrics Defined

Metric Description
Consistency % Percentage of innings with scores above defined thresholds
Impact Score Weighted score using runs, strike rate, and match context
Boundary % Proportion of runs scored via boundaries
Pressure Performance Index Performance in high-pressure match situations
Volatility Standard deviation of scores to measure consistency

7. Analytical Techniques Used

  • Rolling averages for trend smoothing
  • Year-wise and innings-wise trend analysis
  • Home vs Away vs Neutral comparison
  • Dismissal pattern analysis
  • Forecasting future performance (analytical projection)

8. Sample Analytical Logic

Rolling Average (Last 10 Innings) =
AVERAGEX(
  TOPN(
    10,
    FILTER(
      ALL(Innings),
      Innings[Innings_Number] <= MAX(Innings[Innings_Number])
    ),
    Innings[Innings_Number], DESC
  ),
  Innings[Runs]
)

9. Dashboard Structure

  • Overview: Career summary & key indicators
  • Consistency Analysis: 30+/50+/100+ distribution
  • Conditions: Home vs Away vs Neutral performance
  • Dismissals: Mode of dismissal impact
  • Trends & Forecasting: Moving averages & projections
  • Insights: Data-driven observations

10. Insights Generated

  • Higher consistency in home conditions compared to away
  • Performance volatility varies significantly across formats
  • Strike rate plays a larger role in impact than raw runs
  • Not-out innings heavily influence impact metrics
